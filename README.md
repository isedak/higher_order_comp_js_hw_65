# Приложение с созданным higher-order component

**Задание:**

Создайте HOC, который будет следить за любыми запросами внутри компонента и показывать спиннер (loader) в случае сетевой активности. Когда запрос заканчивается, спиннер должен убраться.

Сделайте какое-нибудь простое демонстрационное приложение, которое будет использовать ваш компонент.

Например, это может быть одна страница с загружаемыми элементами чего-то, что вы делали раньше (цитаты, посты, заказы и т.п.)
